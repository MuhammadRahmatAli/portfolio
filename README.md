## âš™ï¸ Build & Deploy

- Install dependencies:

```bash
npm install
```

- Build the compiled Tailwind CSS (writes to `src/output.css`):

```bash
npm run build:css
```

- During development, watch for changes to regenerate `src/output.css`:

```bash
npm run watch:css
```

- To deploy automatically to GitHub Pages, push to the `main` branch. A GitHub Actions workflow will build the CSS and publish the `src/` folder to the `gh-pages` branch.
